Proyecto Respaldo Academico

Mi repositorio personal de respaldo académico. Aquí documento mi evolución, organizo mis apuntes y guardo las soluciones a los retos planteados a lo largo de mi formación.

Este espacio funciona como mi bitácora de aprendizaje. El objetivo es mantener un orden riguroso de los conocimientos adquiridos, facilitando la consulta rápida y el intercambio de ideas con compañeros.

Tecnologías y Herramientas

Dependiendo del curso, en este repositorio verás implementaciones usando:

Lenguajes: (Ej: Python, C++, Java - Completa según lo que uses)

Herramientas de Diseño: (Ej: Lucidchart, Figma, Cisco Packet Tracer).
